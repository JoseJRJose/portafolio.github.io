Restaurante - Bar "El Arca"


Este proyecto fue realizado como parte de una evaluación estudiantil en la universidad, con el objetivo de crear una página web para un restaurante bar llamado El Arca. La idea era desarrollar una plataforma interactiva y funcional que mejorara la experiencia del cliente y permitiera una gestión sencilla de reservas y menús.

A continuación, explico cómo funcionaba la página cuando estaba alojada en un host con base de datos externa, así como las limitaciones que surgen al subirla sin una base de datos.

Funcionalidades Completas en el Host con Base de Datos Externa:

  • Registro de Usuarios: Permite registro con correo, nombre, teléfono y contraseña. Validación de datos.
  • Recuperación de Contraseña: Envío de instrucciones para restablecer la contraseña.
  • Inicio de Sesión: Acceso con correo y contraseña, validación de credenciales.
  • Menú Principal: Carrusel de imágenes del restaurante y su información.
  • Menú Interactivo: Categorías con productos, precios e imágenes ilustrativas.
  • Reservaciones: Formulario con selección de zona, fecha y hora, y confirmación por correo.
  • Perfil de Usuario: Información personal y reservaciones activas.
  • Cerrar Sesión: Opción para salir y volver a la pantalla de inicio.

Limitaciones al Subir el Proyecto a un Host sin Base de Datos:

  • Registro de Usuarios: No se pueden crear nuevas cuentas ni validar correos.
  • Recuperación de Contraseña: No disponible sin base de datos.
  • Acceso con Credenciales: Solo con datos preconfigurados.
  • Menú Principal y Reservaciones: Sin confirmaciones por correo.
  • Perfil de Usuario: No se pueden almacenar ni actualizar datos.
  • Otras Funcionalidades: Sin almacenamiento dinámico ni interacciones avanzadas.

En resumen, este proyecto demostró cómo una página web con base de datos puede mejorar la experiencia del usuario, permitiendo una gestión completa de clientes, reservas y menús. Sin embargo, al alojarlo sin base de datos, muchas funcionalidades se ven limitadas, reduciendo su interactividad.
